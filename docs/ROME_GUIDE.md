# Rome city guide

Same model as `paris/`: English titles (`name_en`), Italian subtitle
(`subtitle_it`), coat and flag on the title page, Commons-backed places
(favouring façades and public space; St. Peter's entry uses an exterior
photo only).

- **Scripts:** `download_rome_images.py`, `validate_rome_sources.py`,
  `build_rome_pdf.py` (`--html-only` for HTML without Playwright PDF).
- **Images:** committed under `rome/images/` for reproducibility.
