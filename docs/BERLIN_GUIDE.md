# Berlin city guide

Same model as `budapest/`: English titles (`name_en`), German subtitle
(`subtitle_de`), coat and flag on the title page, 20 Commons-backed places.

- **Scripts:** `download_berlin_images.py`, `validate_berlin_sources.py`,
  `build_berlin_pdf.py` (`--html-only` for HTML without Playwright PDF).
- **Images:** committed under `berlin/images/` for reproducibility.
