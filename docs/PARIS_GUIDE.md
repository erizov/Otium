# Paris city guide

Same model as `berlin/`: English titles (`name_en`), French subtitle
(`subtitle_fr`), coat and flag on the title page, Commons-backed places.

- **Scripts:** `download_paris_images.py`, `validate_paris_sources.py`,
  `build_paris_pdf.py` (`--html-only` for HTML without Playwright PDF).
- **Images:** committed under `paris/images/` for reproducibility.
