# Barcelona city guide

Same model as `rome/`: English titles (`name_en`), Catalan subtitle
(`subtitle_ca`), coat and flag on the title page, ~20 Commons-backed places
(Gaudí works, Old City, Montjuïc, waterfront).

- **Scripts:** `download_barcelona_images.py`, `validate_barcelona_sources.py`,
  `build_barcelona_pdf.py` (`--html-only` for HTML without Playwright PDF).
- **Images:** committed under `barcelona/images/` for reproducibility.
